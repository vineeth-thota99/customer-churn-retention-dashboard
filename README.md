# Customer Churn Retention Dashboard

## Tools Used
Power BI, PostgreSQL, SQL, DAX

## Objective
Analyze telecom customer churn and identify retention opportunities.

## Key KPIs
- Total Customers: 7043
- Churned Customers: 1869
- Churn Rate: 26.54%

## Key Insights
- Highest churn among Month-to-month contracts
- Electronic check customers churn most
- Lower tenure linked to churn
